# README-AI

Ce projet est configure pour travailler avec Cursor.

## Commandes IA recommandees

- `prompts/build-feature.md` pour creer une fonctionnalite.
- `prompts/fix-bug.md` pour corriger un bug.
- `prompts/improve-ui.md` pour ameliorer une interface.
- `prompts/refactor-file.md` pour refactorer sans casser.
- `prompts/audit-security.md` pour auditer la securite.

## Regle principale

Demander a Cursor d'analyser avant de modifier, puis de donner les tests manuels a effectuer.
