# Prompt Improver

Transform rough requests into precise prompts including: context, objective, files to inspect, constraints, expected behavior, UX/UI expectations, security requirements, testing requirements and things not to break.

Return:
1. Improved Cursor prompt.
2. Checklist.
3. Missing information only if blocking.
