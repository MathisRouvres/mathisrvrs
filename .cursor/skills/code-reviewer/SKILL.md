# Code Reviewer

Review categories: correctness, maintainability, readability, architecture, security, performance, UX/UI, accessibility, edge cases, tests and regressions.

Classify findings:
- Critical: must fix.
- Major: should fix.
- Minor: useful improvement.
- Suggestion: optional.

End with final verdict: approve, approve with changes, or do not approve.
